# 📋 Formulario de Registro de Usuario

## 📌 Descripción
Este proyecto consiste en el desarrollo de una página web que contiene un formulario de registro de usuario. Fue realizado utilizando **HTML, CSS, Bootstrap y JavaScript**, aplicando validaciones y mostrando los datos ingresados por el usuario.

## 🎯 Objetivo
Desarrollar una interfaz web interactiva que permita:
- Registrar datos de un usuario
- Validar los campos del formulario
- Mostrar un mensaje con la información ingresada

## 🛠️ Tecnologías utilizadas
- **HTML5** → Estructura del formulario  
- **CSS3** → Estilos personalizados  
- **Bootstrap** → Diseño responsive y componentes visuales  
- **JavaScript** → Validación y manipulación de datos  

## 🧩 Funcionalidades
✔ Formulario con los siguientes campos:
- Nombre  
- Apellido  
- Correo electrónico  
- Edad  
- Contraseña  

✔ Validaciones:
- Todos los campos son obligatorios  
- Verificación de campos vacíos  

✔ Interacción:
- Captura de datos ingresados  
- Mensaje de confirmación en pantalla  

## 💻 Estructura del proyecto
- **login.html**
- **estilos.css**
- **app.js**

## 📚 Aprendizajes

**En este proyecto se aplicaron conceptos como:**

- Creación de formularios en HTML
- Uso de Bootstrap para diseño
- Validación de datos con JavaScript
- Manipulación del DOM
